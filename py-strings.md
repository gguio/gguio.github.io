# Шпаргалка: Срезы и методы строк в Python

В этом документе собраны основные приемы работы со строками: получение срезов, поиск, подсчет вхождений и изменение текста.

---

## 1. Срезы (Slicing)

Срезы позволяют получить часть строки или списка.
Синтаксис: `объект[start:stop:step]`

- **start**: Индекс начала (включительно).
- **stop**: Индекс конца (**не включая** его).
- **step**: Шаг (необязательно).



### Примеры:

```python
s = "Hello Python"

# 1. С начала до 5-го индекса
print(s[0:5])   # Результат: 'Hello'

# 2. С 6-го индекса до конца
print(s[6:])    # Результат: 'Python'

# 3. Разворот строки (шаг -1)
print(s[::-1])  # Результат: 'nohtyP olleH'
```

## 2. Метод .count()

Используется для подсчета количества вхождений подстроки в строку.
Синтаксис: `строка.count(что_ищем)`

### Примеры:

```python
text = "банан и ананас"

# Считаем, сколько раз встречается "ан"
print(text.count("ан"))  # Результат: 4
```

## 3. Метод .replace()

Заменяет одну подстроку на другую.
Синтаксис: `строка.replace(старое, новое, сколько_раз)`

### Примеры:

```python
msg = "Java — это круто. Я учу Java."
print(msg.replace("Java", "Python")) # Заменит всё
```

## 4. Методы поиска: find и index

Эти методы ищут подстроку и возвращают индекс её первого вхождения.

| Метод | Если подстрока **найдена** | Если подстрока **НЕ найдена** |
| :--- | :--- | :--- |
| `.find(sub)` | Возвращает наименьший индекс | Возвращает `-1` |
| `.index(sub)` | Возвращает наименьший индекс | Вызывает ошибку `ValueError` |

### Примеры поиска:

```python
s = "Программирование"

# Поиск слева направо
print(s.find("амм"))   # Результат: 5
print(s.find("Java"))  # Результат: -1

# Поиск справа налево (r = right)
print(s.rfind("о"))    # Результат: 13 (последняя 'о')

# Использование index (аналогично find, но с ошибкой в конце)
print(s.index("Про"))  # Результат: 0
```

## 5. Проверка и изменение регистра

Методы для работы с размером букв и типом содержимого.

```python
s = "python 3.10"

print(s.upper())      # "PYTHON 3.10" (в верхний регистр)
print(s.lower())      # "python 3.10" (в нижний регистр)
print(s.capitalize()) # "Python 3.10" (первая буква заглавная)
print(s.title())      # "Python 3.10" (каждое слово с заглавной)

# Проверки (возвращают True или False)
print("123".isdigit())    # True (только цифры)
print("abc".isalpha())    # True (только буквы)
print(s.startswith("py")) # True (начинается с...)
print(s.endswith("10"))   # True (заканчивается на...)
```

## 6. Удаление пробелов: strip

Используется для очистки строки от лишних невидимых символов по краям.

```python
raw_str = "   текст с пробелами   "
print(raw_str.strip())  # "текст с пробелами"
print(raw_str.lstrip()) # "текст с пробелами   " (только слева)
print(raw_str.rstrip()) # "   текст с пробелами" (только справа)
```
